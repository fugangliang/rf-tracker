## 変更・デプロイ手順

1. `docs/` を編集 → `node test/acceptance.js`（34項目、data/があれば受け入れ含む）
2. **`docs/sw.js` の `VERSION` を上げる**（忘れるとクライアントのキャッシュが更新されない）
3. commit → push → Pagesに自動反映（約15秒）
- ghトークンに workflow スコープなし。`.github/workflows/` はpush不可（ブランチ配信を採用した理由）
